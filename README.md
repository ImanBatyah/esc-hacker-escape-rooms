[Esc] Hacker Escape Rooms – Mobilmeny

Det här är min inlämningsuppgift i kursen JavaScript Grund. Uppgiften gick ut på att bygga en mobilanpassad meny till startsidan för [Esc] Hacker Escape Rooms. Jag har gjort två olika versioner av menyn enligt instruktionen.

Struktur

variant-a-static-html – en statisk version som jag byggt med vanlig HTML och CSS

variant-b-dom-api – en version där menyn skapas med JavaScript och DOM API

Publicerad sida

Jag har publicerat den statiska versionen (variant A) på GitHub Pages:
https://imanbatyah.github.io/esc-hacker-escape-rooms/

GitHub Pages kan bara visa en branch i taget, så därför är bara variant-a publicerad. Variant-b finns i repot för granskning.
