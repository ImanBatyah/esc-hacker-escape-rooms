[Esc] Hacker Escape Rooms – Mobilmeny
Det här projektet är min inlämningsuppgift där jag har byggt en mobilanpassad meny till startsidan för [Esc] Hacker Escape Rooms.
Menyn ska bara synas på små skärmar och öppnas med en knapp (hamburgermenyn).

Struktur
Jag har gjort uppgiften på två olika sätt:

variant-a-static-html – en statisk HTML-version

variant-b-dom-api – en version där menyn skapas med JavaScript

Variant A – Statisk HTML

I den här versionen ligger menyn direkt i HTML-filen och styrs med CSS för utseende och animationer.
JavaScript används bara för att öppna och stänga menyn.
På små skärmar visas en hamburgerknapp som öppnar en fullskärmsmeny.


Variant B – DOM API

Här byggs hela menyn upp med JavaScript istället för att ligga i HTML.
JavaScript skapar alla delar av menyn när sidan laddas, inklusive knappar och länkar.
Funktionen är densamma som i variant A, men strukturen ligger helt i JavaScript

Tekniker jag använt
HTML, CSS, JavaScript, Git, GitHub och GitHub Pages.
