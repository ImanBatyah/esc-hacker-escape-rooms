[Esc] Hacker Escape Rooms – Mobilmeny

Det här är min inlämningsuppgift i kursen JavaScript Grund. Jag har byggt en mobilanpassad meny till startsidan för [Esc] Hacker Escape Rooms. Menyn visas bara på små skärmar och öppnas med en hamburgermeny.

Jag har gjort två versioner av lösningen: en statisk HTML-version och en version där menyn byggs upp med JavaScript och DOM API. Här nedan kan du välja vilken variant du vill titta på.

Variant A – statisk HTML

I den här versionen ligger menyn direkt i HTML-filen och styrs med CSS för utseende och animationer. JavaScript används bara för att öppna och stänga menyn.

Öppna variant A

Variant B – DOM API (JavaScript)

I den här versionen byggs mobilmenyn upp med JavaScript när sidan laddas. Menyn skapas alltså med DOM API i stället för att ligga färdig i HTML-filen.

Öppna variant B
